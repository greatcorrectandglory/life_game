# 变更日志 (CHANGELOG)

## [1.1.0] - 2025-01-06

### 新增 🎉
- **认知吸收率系统**：根据信息熵和认知负荷差异化行动收益
- **艾宾浩斯衰减机制**：重复行动会导致技能收益衰减
- **技能标签系统**：支持技能跨行业迁移
- **存档系统**：支持手动存档、自动存档、导入导出
- **日志系统**：完善的日志记录和调试功能
- **响应式状态管理**：支持路径订阅和批量更新

### 改进 ✨
- **代码架构重构**：拆分 main.js 为多个模块
- **配置数据提取**：将 ACTIONS、SKILL_TREE、CHAPTERS 等提取到独立文件
- **错误处理**：完善的异常捕获和错误日志
- **工具函数库**：添加防抖、节流、深克隆等工具函数
- **类型安全**：添加 JSDoc 类型注释

### 修复 🐛
- 优化状态更新性能，减少不必要的渲染
- 修复内存泄漏问题

### 文档 📝
- 新增 README.md 项目说明
- 新增 DEVELOPMENT.md 开发指南
- 新增 OPTIMIZATION_SUGGESTIONS.md 优化建议
- 新增 CHANGELOG.md 变更日志

### 开发工具 🔧
- 添加 Vite 构建配置
- 添加 Vitest 测试框架
- 添加 ESLint 代码检查
- 添加 Prettier 代码格式化
- 添加 .gitignore 配置
